 Les eco-organismes assurent leurs missions dans l'objectif de couvrir l'ensemble du territoire francais y compris les departements d'outre-mer et les collectivites d'outre-mer ou le code de l'environnement s'applique.Dans un souci d'equite, les eco-organismes metropolitains assurent un soutien technique aux eco-organismes situes dans les departements d'outre-mer et dans les collectivites d'outre-mer ou le code de l'environnement s'applique.
