Les éco-organismes assurent leurs missions dans l'objectif de couvrir l'ensemble du territoire français y compris les départements d'outre-mer et les collectivités d'outre-mer où le code de l'environnement s'applique.Dans un souci d'équité, les éco-organismes métropolitains assurent un soutien technique aux éco-organismes situés dans les départements d'outre-mer et dans les collectivités d'outre-mer où le code de l'environnement s'applique.
