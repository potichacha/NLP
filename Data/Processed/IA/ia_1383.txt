Cara membuat config mikrotik

Untuk membuat config (konfigurasi) MikroTik, ikuti langkah-langkah berikut: Buka winbox dan log in ke MikroTik. Pilih menu IP dan klik pada-address. Klik tombol add (+) dan masukkan IP Address, Subnet Mask dan Default Gateway. Untuk mengkonfigurasi jaringan lokal, pilih menu DHCP Server. Klik tombol setup dan masukkan IP Pool, Range DHCP, dan Default Gateway. Konfigurasi DNS Server pada menu IP > DNS dengan menambahkan DNS Server. Pilih menu Firewall dan atur kebijakan firewall sesuai dengan kebutuhan. Misalnya, membuat rule block port yang tidak digunakan. Konfigurasi Wi-Fi pada menu Wireless. Klik tombol add (+) dan masukkan SSID, security profile dan password. Jika ingin membatasi akses internet untuk pengguna yang terhubung ke jaringan, buatlah hotspot user manager pada menu IP > Hotspot. Terakhir, Simpan konfigurasi MikroTik dengan klik tombol Save pada menu File. Itulah sedikit langkah-langkah membuat konfigurasi dasar pada MikroTik. Penting untuk diingat bahwa setiap jaringan memiliki konfigurasi yang berbeda tergantung pada kebutuhan dan lingkungan.
